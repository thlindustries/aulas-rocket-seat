# Servidor básico node para praticar configurações do git.

## Instalação:

Use o [npm](https://www.npmjs.com/) para instalar as dependências.

```bash
cd /aula-git
npm install
npm start
```

# Acessar a aplicação:
Para acessar a aplicação basta entrar no endereço:

[localhost:3000](http://localhost:1234)
